# 简述     
- 一个最简单 vuex 分层小例子       
- 实现中文姓名、英文姓名混排排序   
- 实现排序后的姓名列表，下载导出为 Excel 文件   


# grocery
- 使用的相关仓库     

```bash
# 汉字转拼音
# https://github.com/hotoo/pinyin
npm install pinyin

# 下载导出为 Excel 文件 
# https://github.com/SheetJS/sheetjs
npm install xlsx
```
